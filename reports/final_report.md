## 项目简介

本项目旨在对一个包含两家酒店（城市酒店和度假酒店）的预订数据集进行深入的探索性数据分析（EDA）。分析的目标是理解客户预订行为、识别关键业务模式，并最终提出三个具有商业价值的营销问题，以帮助酒店改善其市场表现和运营效率。此分析将涵盖数据清洗、变量探索、关系挖掘和可视化呈现。

---

## 第一部分：数据探索与发现

我们对包含 **119,390** 条预订记录的数据集进行了全面的探索性分析。以下是核心发现的总结。

### 数据概况与关键指标

经过初步的数据清洗（处理缺失值和异常值）后，我们提炼出以下关键业务指标，它们共同描绘了酒店运营的基本面貌。

| 指标 | 数值 |
| :--- | :--- |
| 酒店类型分布 | 城市酒店: 66.4%, 度假酒店: 33.6% |
| 总体取消率 | 37.04% |
| 平均提前预订天数 | 104 天 |
| 平均住宿时长 | 3.4 晚 |
| 回头客比例 | 3.19% |

这些指标揭示了两个亟待解决的核心挑战：**高企的预订取消率**和**极低的客户忠诚度**。

### 公共数据集的整合价值

为了将酒店的微观表现与宏观经济背景相结合，我们引入了来自**世界银行**、**葡萄牙国家统计局**和**Eurostat**的公共数据集。这些数据（如CPI、旅游业统计数据）能够帮助我们验证季节性趋势、评估定价策略的合理性，并为市场营销决策提供宏观经济层面的支持。

### 可视化分析洞察

我们从四个维度进行了可视化分析，以揭示数据背后的深层故事。

#### 1. 取消率分析

预订取消是侵蚀酒店利润的主要因素。分析显示，城市酒店的取消率（41.7%）远高于度假酒店（27.8%）。一个关键发现是，取消风险与预订的提前期（Lead Time）呈正相关，提前时间越长，取消的可能性越大。这凸显了对长期预订进行风险管理的必要性。

![取消率分析图](/home/ubuntu/cancellation_analysis.png)

#### 2. 收入 (ADR) 分析

平均每日房价（ADR）表现出强烈的季节性，夏季（尤其是8月）为价格高峰，而冬季则为低谷。值得注意的是，通过“在线旅行社”（Online TA）和“直接预订”（Direct）渠道获取的客户，其ADR最高，这表明客户愿意为预订的便捷性或与酒店的直接沟通支付溢价。

![收入分析图](/home/ubuntu/revenue_analysis.png)

#### 3. 预订模式分析

预订量同样遵循季节性规律，夏季是预订旺季。大部分客户为短途旅行者，住宿时长集中在1-4晚。回头客的行为模式与新客户存在显著差异：他们的取消率更低，证明了其更高的忠诚度和预订确定性。

![预订模式分析图](/home/ubuntu/booking_patterns.png)

#### 4. 客户来源分析

酒店的主要客户群体来自欧洲，其中葡萄牙、英国和法国位列前三。在分销渠道方面，酒店高度依赖第三方，超过80%的预订来自旅行社和旅游运营商（TA/TO），其中线上渠道占据主导地位。这提示酒店需要思考如何平衡渠道依赖与盈利能力。

![客户来源分析图](/home/ubuntu/customer_source_analysis.png)

## 第二部分：三个核心营销问题

基于以上数据洞察，我们提炼出以下三个具有战略意义的营销问题，旨在为酒店提供具体、可行的改进方向。

### 问题一：如何有效降低高企的预订取消率，特别是针对城市酒店和长期预订？

**问题背景**: 高达37%的总体取消率，尤其是城市酒店的41.7%，是酒店面临的严峻挑战。长期预订的高不确定性进一步加剧了收入损失和运营困难。

**分析与建议**: 我们建议实施差异化的押金策略，例如对提前期较长的预订采用阶梯式或更高的预付比例，以增加客户的承诺成本。同时，提供灵活的改期选项而非单一的取消路径，可以有效保留部分潜在流失的客户。此外，利用自动化工具在客户入住前进行有意义的互动（如发送欢迎邮件、提供升级选项），也能显著增强客户的预订粘性，从而降低取消率。

### 问题二：如何显著提升客户忠诚度，将当前仅为3.19%的回头客比例提高？

**问题背景**: 极低的回头客比例（3.19%）表明酒店在客户关系维护和忠诚度培养方面存在巨大潜力。鉴于回头客是更稳定、更高质量的收入来源，提升这一比例对于酒店的长期健康发展至关重要。

**分析与建议**: 建立一个结构化的客户忠诚度计划是核心。该计划应为回头客提供明确且有吸引力的激励，如积分兑换、会员专属折扣或免费升房等。在客户离店后，通过个性化的邮件营销保持沟通，并根据其过往偏好推送相关优惠，能有效驱动二次预订。根本上，建立一个闭环的客户反馈系统，系统地收集、分析并响应客户的意见，是持续优化服务、提升客户满意度和忠诚度的基石。

### 问题三：如何优化动态定价和渠道管理策略，以最大化整体收入？

**问题背景**: 酒店收入在季节间的剧烈波动，以及对高佣金分销渠道的过度依赖，是收入优化的两大障碍。如何在保证入住率的同时最大化ADR，并将客户引向利润更高的直销渠道，是核心议题。

**分析与建议**: 我们建议采用更精细化的动态定价模型，该模型应综合考虑预订提前期、入住时长、客户细分、季节性以及宏观经济指标（如CPI）。在需求疲软的淡季，应创新地推出捆绑式套餐产品（如“住宿+餐饮”、“住宿+本地文化体验”），以非价格竞争手段吸引客户。同时，必须大力发展直接预订渠道，通过提供“官网最优价保证”、会员专享套餐等策略，逐步将客户从OTA平台转移至利润更高的自有渠道。
